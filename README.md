# Online Retail Sales Analysis (2010-2011)
This project is from UCL Machine Learning Repository. Using a random sample of 1,685 of cutomers worlwide. The data was cleaned with Excel, processed using SQL, and visualised in Tableau to answer the 4 key business questions. 
# Tools used
- Excel (data cleaning)
- SQL (data analysis and queries)
- Tableau ( data visualization)
# Business questions answered:
1) Which products generate the most revenue?
2) Which countries generate the most sales?
3) Which months have the highest sales?
4) Which products sell the highest quantity?
# SQL Analysis 
All SQL analysis performed using SQL queries, in SQL folder. 
# Tableau Dashboard 
The full interactive dashboard and story can be viewed here: https://public.tableau.com/app/profile/tony.mccutcheon/vizzes
# Key Insights:
- The UK makes the most sales, outpreforming all other world wide countries.
- Revenue is concentrated in a small numbers of countries.
- Sales show seasonal fluctuations over time.
- A small number of products drive the majority of sales.
