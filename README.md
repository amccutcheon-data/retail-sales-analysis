# online_retail-sales-analysis
Tools used: Excel, SQL, and Tableau.
Business questions answered:
1) Which products generate the most revenue?
2) Which countries generate the most sales?
3) Which months have the highest sales?
4) Which productssell the highest quantity?
   
Insights: A small number of products generate the majority of revenue. Sales are highly concentrated in just a few countries, and some months show seasonal spikes.
